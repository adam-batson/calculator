Calculator

Project GitHub

This is a calculator made with HTML, CSS, and JavaScript for Microsoft Leap Internal project.
The calculator takes a number, then an operation, then a second number and finds the result.

    To use:
  1. Click a number's digits OR type a number into the yellow box.
  2. Click the operation to be performed.
  3. Repeat step 1.
  4. Click = for the result.
    Notes:
  • The +/- button will switch the number between positive and negative.
  • The √ button will get the square root of the number.
  • The x2 button will square the number.
  • The C button resets the numbers.
  • Each +, -, x, or / operation must be followed by = before another can be performed.
